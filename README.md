**ayu** is a bright color theme and comes in three versions — _dark_, _mirage_, and _light_ — for all-day comfortable work. Handcrafted for Xcode.

## Install

With fonts:

```sh
curl -fsSL https://raw.githubusercontent.com/ayu-theme/ayu-xcode/master/install.sh | sh -s -- --with-fonts
```

Without fonts:

```sh
curl -fsSL https://raw.githubusercontent.com/ayu-theme/ayu-xcode/master/install.sh | sh -s -- --no-fonts
```

Restart Xcode and select the theme in Preferences > Themes.
